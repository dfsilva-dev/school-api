# Escola API

## Descrição

API criada para possibilitar o cadastro de cursos, estudantes e matriculas, com o intuito de praticar o uso da biblioteca Django Rest Framework e o processo de documentação utilizando o Swagger.

## Description

This API was created to enable the registration of courses, students, and enrollments, with the aim of practicing the use of the Django Rest Framework library and the documentation process using Swagger.
